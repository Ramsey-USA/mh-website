# Legacy Audit Utilities

This folder consolidates one-off and exploratory audit/automation scripts that were previously stored at the repository root.

## Purpose

- Keep the root directory focused on active project entry points.
- Preserve historical and ad-hoc utilities without deleting them.
- Provide a single location for manual audit tooling.

## Scripts

- `audit_localization.py`
- `auditor.js`
- `regenerate_report.sh`
- `report_gen.sh`
- `route_audit.mjs`
- `scanner.sh`
- `sweep_locales.sh`
- `update_files.py`

## Usage Notes

- Run these scripts from the repository root unless the script itself documents otherwise.
- Most scripts are exploratory and are not part of CI/CD.
- Before using a script for production changes, review it for current paths and assumptions.
