# Social Image Index

This folder holds social media image assets and reusable post templates.

## Current Contents

- [announcements/](./announcements/) - announcement graphics and social post variants
- [templates/](./templates/) - reusable social post templates
- [testimonials/](./testimonials/) - testimonial graphics and quote-card assets

## Tracking Tips

- Keep social campaigns and reusable templates in separate subfolders.
- If a post family grows beyond a few files, give it its own dedicated folder.
- Treat the `.gitkeep` files here as placeholders until real assets are added.
