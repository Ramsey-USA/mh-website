# Safety Image Index

This folder holds safety-related image assets used across the website.

## Current Contents

- [compliance.webp](./compliance.webp) - compliance-focused hero or section image
- [emr-awards.webp](./emr-awards.webp) - EMR and awards graphic
- [quality-control.webp](./quality-control.webp) - quality control graphic
- [safety-culture.webp](./safety-culture.webp) - safety culture graphic
- [snapshots/](./snapshots/) - placeholder workspace for captured or comparison images

## Tracking Tips

- Keep final-use assets at the top level and transient captures in `snapshots/`.
- Add new safety visuals here only when they are reusable website assets.
- If a safety image family grows, split it into a dedicated subfolder with its own README.
