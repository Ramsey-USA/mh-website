# SEO Section Ordering Reference

**Category:** Technical - SEO Reference
**Last Updated:** July 10, 2026
**Status:** Active

This reference captures page-specific section ordering patterns used by lead-path routes.
Use it with [Complete SEO & Search Optimization Guide](./seo-complete-guide.md).

## Ordering Standard

All lead-path pages follow: **Discover → Trust → Proof → Action**.

Canonical standard:

- [Universal Page Flow Standard](../../development/standards/universal-page-flow-standard.md)

## Homepage ✅

1. Hero _(Discover)_
2. Verified proof cards _(Trust/Proof)_
3. Service overview cards + service-route chips _(Discover)_
4. Featured public project case studies _(Proof)_
5. Why MH planning/communication/execution points _(Trust)_
6. TestimonialsSectionDeferred _(Proof)_
7. NextStepsSection _(Action)_

## Services Hub ✅

1. ServicesShowcase intro copy _(Discover)_
2. Delivery Path selector _(Discover)_
3. Project Focus selector _(Discover)_
4. Specific Service Cards _(Discover)_
5. Service detail modal + contact CTA _(Action)_
