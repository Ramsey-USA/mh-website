### Example Implementation

```tsx

  icon="military_tech"

/>
```

## Technical Details

### Component Props

```typescript
interface MaterialIconProps {
  icon: string; // Material icon name
  size?: "xs" | "sm" | "md" | "lg" | "xl" | "2xl" | "3xl" | "4xl" | "5xl";
  theme?: "veteran" | "military" | "tactical" | "default";
  ariaLabel?: string; // Accessibility label
  interactive?: boolean; // Hover effects
  className?: string; // Additional styling
}
```

### Theme Colors

```css
veteran: #BD9264  /* Bronze */
military: #386851 /* Hunter Green */
tactical: tan     /* Field Tan */
default: gradient /* Brand colors */
```

## Maintenance Guidelines

### When Adding New Icons

1. **Check consistency** - Use existing icon patterns first
2. **Apply themes** - Use military or veteran theme appropriately
3. **Add ariaLabel** - Always include accessibility label
4. **Document usage** - Update this guide if introducing new patterns
5. **Validate TypeScript** - Ensure zero errors

### When Updating Icons

1. **Review cross-page usage** - Check if icon appears on other pages
2. **Maintain consistency** - Don't break established patterns
3. **Test accessibility** - Verify screen reader support
4. **Update documentation** - Keep this guide current

### When Removing Icons

1. **Check dependencies** - Search for all usages
2. **Verify replacements** - Ensure alternative icon maintains meaning
3. **Test all pages** - Validate no broken references
4. **Update documentation** - Remove from this guide
