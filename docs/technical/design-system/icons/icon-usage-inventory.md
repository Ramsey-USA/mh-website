## üìê Design Guidelines

### 1. Unique Context Principle

**Each icon should represent a unique concept or page** to avoid user confusion and maintain clear mental models.

**‚úÖ Good Example:**

- `people` for Our Team (internal staff)
- `business` for Trade Partners (external vendors)

**‚ùå Bad Example:**

- Using `people` for both internal team AND external partners (confusing)

---

### 2. Semantic Meaning

**Icons should clearly relate to their associated content** and be instantly recognizable.

**‚úÖ Good Example:**

- `calculate` for AI Estimator (math/calculation connection)
- `account_balance` for Government (government building connection)

**‚ùå Bad Example:**

- `star` for Government (no clear connection)
- `settings` for AI Estimator (generic, not specific)

---

### 3. Consistent Style

**All icons use Google Material Design icon system** for visual consistency.

**Requirements:**

- Material Icons library exclusively
- Consistent sizing (xs, sm, md, lg, xl)
- Theme-aware colors (light/dark mode)
- No custom SVG icons unless approved exception
- No emoji characters in source code

**Reference:** [Icon Policy Complete](./icon-policy-complete.md)

---

### 4. Accessibility

**Icons must be accompanied by descriptive text or aria-labels** for screen readers.

**‚úÖ Good Example:**

```tsx
// With visible text label
<button>
  <MaterialIcon icon="event" className="mr-2" />
  Schedule Consultation
</button>

// Icon-only with aria-label
<button aria-label="Close menu">
  <MaterialIcon icon="close" />
</button>
```

**‚ùå Bad Example:**

```tsx
// No context for screen readers
<button>
  <MaterialIcon icon="event" />
</button>
```

---

### 5. Visual Hierarchy

**Important actions use more prominent icons** with appropriate sizing and placement.

**Primary Actions:** Large icons (`lg` or `xl`)  
**Secondary Actions:** Medium icons (`md`)  
**Tertiary Actions:** Small icons (`sm` or `xs`)

**Example:**

```tsx
// Primary CTA - Large icon
<Button variant="primary" size="lg">
  <MaterialIcon icon="handshake" size="lg" className="mr-3" />
  Start Partnership
</Button>

// Secondary CTA - Medium icon
<Button variant="secondary" size="md">
  <MaterialIcon icon="calculate" size="md" className="mr-2" />
  Try AI Estimator
</Button>

// Tertiary link - Small icon
<Link href="/contact">
  <MaterialIcon icon="arrow_forward" size="sm" className="ml-1" />
</Link>
```
