## üé® Styling Standards

### Color Application

**Military Context (Veteran-Owned):**

```tsx
// Veteran badges and military content
className = "text-bronze-300";
className = "text-brand-accent";

// Army (primary veteran status)
className = "text-brand-primary";

// Navy
className = "text-brand-secondary";

// Multi-branch recognition
className = "text-purple-600";
```

**Tactical/Mission Context:**

```tsx
// Mission-critical actions
className = "text-brand-primary";

// Strategic planning
className = "text-blue-600 dark:text-blue-400";

// Federal/Government
className = "text-gray-700 dark:text-gray-300";
```

**Service Categories:**

```tsx
// Construction services
className = "text-brand-primary";

// Emergency/urgent
className = "text-orange-600 dark:text-orange-400";

// Success/completion
className = "text-green-600 dark:text-green-400";
```

---

### Size Guidelines

**Hero Sections:**

```tsx
size = "4xl"; // 96px - Hero icons, major focal points
size = "3xl"; // 72px - Large feature icons
```

**Section Headers:**

```tsx
size = "2xl"; // 60px - Section header icons (SectionHeader component)
size = "xl"; // 48px - Large card headers
```

**Cards & Features:**

```tsx
size = "xl"; // 48px - Primary card icons
size = "lg"; // 36px - Standard feature icons
```

**Buttons & Inline:**

```tsx
size = "lg"; // 36px - Large buttons
size = "md"; // 30px - Standard buttons
size = "sm"; // 24px - Inline text, small buttons
```

---

### Animation Patterns

**Military Precision (Subtle, Controlled):**

```tsx
// Standard hover (recommended for most icons)
className = "group-hover:scale-110 transition-transform duration-300";

// Tactical rotation (use sparingly)
className = "group-hover:rotate-12 transition-transform duration-300";

// Mission focus (pulsing attention)
className = "animate-pulse";

// Status indicator (continuous rotation)
className = "animate-spin";
```

**Examples:**

```tsx
// Badge with military precision hover
<div className="group">
  <MaterialIcon
    icon="military_tech"
    size="xl"
    className="text-bronze-300 group-hover:scale-110 transition-transform duration-300"
  />
</div>

// Mission planning icon with tactical rotation
<div className="group">
  <MaterialIcon
    icon="map"
    size="2xl"
    className="text-brand-primary group-hover:rotate-12 transition-transform duration-300"
  />
</div>
```

## üéØ Key Principles

1. **Military-Inspired, Not Military-Only**: Icons should resonate with veterans while remaining accessible to all clients
2. **Professional First**: Construction industry professionalism comes before military themes
3. **Semantic Clarity**: Icons should clearly communicate their purpose
4. **Consistent Application**: Use the same icon for the same purpose site-wide
5. **Accessibility**: All icons must meet accessibility standards
6. **Performance**: Icon usage should not impact page load times
