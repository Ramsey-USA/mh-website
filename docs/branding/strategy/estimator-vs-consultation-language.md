# AI Estimator vs Human Consultation - Universal Language Guide

**Document Purpose:** Define clear, consistent language that distinguishes our AI Budget Planning Tool from
Human Expert Consultations across all touchpoints.

**Last Updated:** December 14, 2025

---

## üéØ Core Philosophy

**PRIMARY PRINCIPLE:** Human consultation is always the recommended first step. The AI tool is a helpful
optional preparation aid, not a replacement for personal partnership.

**MESSAGING HIERARCHY:**

1. Personal consultation = RECOMMENDED path
2. AI tool = Helpful optional preparation tool
3. Both are free, both serve Client Partners, different purposes

## ‚úÖ Quick Reference Checklist

## üö´ Common Mistakes to Avoid

**DON'T:**

- ‚ùå Position AI tool as better/faster/more convenient than consultation
- ‚ùå Suggest "skip the consultation, just use the tool"
- ‚ùå Call it "automated" (sounds robotic) or "calculator" (sounds basic)
- ‚ùå Imply consultation is only for complex projects
- ‚ùå Present tool as default path with consultation as backup
- ‚ùå Use "meeting" or "appointment" for consultation (too generic/clinical)
- ‚ùå Forget to mention both are free

**DO:**

- ‚úÖ Always recommend consultation first
- ‚úÖ Position tool as helpful preparation aid
- ‚úÖ Emphasize partnership approach in consultation
- ‚úÖ Highlight "free" and "no obligation" for both
- ‚úÖ Use "under 5 minutes" for tool speed
- ‚úÖ Use "comprehensive" for consultation scope
- ‚úÖ Maintain "Building for the Client Partner, NOT the Dollar" philosophy
