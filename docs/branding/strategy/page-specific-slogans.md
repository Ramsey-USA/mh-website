# MH Page Slogans

**Category:** Branding - Strategy
**Last Updated:** July 4, 2026
**Status:** Official Reference

> **Purpose:** This file is the canonical source for unique page-level slogans used across the MH Construction codebase.
> The primary slogan remains [Brand Constants](../brand-constants.md) and every page in scope should have one page-specific slogan in addition to the canonical primary line.

---

## Usage Rules

1. Use one page-specific slogan per page key as the default for that page.
2. A page slogan may be reused on other relevant pages once when it improves congruency.
3. Keep the wording plain, factual, and aligned with MH brand voice.
4. When adding a new page, add its slogan here first, then wire it into the implementation.
5. Prefer wording that naturally echoes the page's SEO name or MH brand name when that improves dual-terminology alignment.

## Reference Note

The page slogans here are intentionally distinct from the supporting slogan family in [Brand Constants](../brand-constants.md) and from the hero-surface coverage map in [Slogan Coverage Matrix](./slogan-coverage-matrix.md).
Use this file when a page needs a page-specific slogan and reuse the same slogan on other relevant pages only when that overlap improves congruency.
When a slogan can naturally include the page's MH brand name, prefer that form over generic wording.
