This folder defines the markdown information architecture for MH branding documentation.

## Routing Rule

Before creating any new branding markdown file:

1. Check whether the content belongs in an existing canonical file.
2. Prefer extending canonical docs instead of creating near-duplicate checklists.
3. If a file must be added, update this governance index and the branding root index.
