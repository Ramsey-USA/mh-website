- âœ… Project/phase is 100% complete
- âœ… Content has been superseded by better documentation
- âœ… Information is purely historical and no longer actionable
- âœ… File hasn't been updated in 6+ months and project is finished

### How to Archive

1. **Move file** to appropriate archive subdirectory
2. **Add archive notice** at top of file:
