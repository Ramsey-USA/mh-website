## Snapshot Context

This file summarizes the original phased rollout plan used to standardize hero
congruency across website routes during the 2026 standards cycle.

## Core Strategy Preserved

- Reuse one congruent hero pattern aligned to the Home baseline.
- Sequence rollout by business impact and trust-critical surfaces first.
- Re-run inventory and quality gates after each phase.
- Require accessibility-readable hero text and no visual regressions on existing routes.

## Historical Phasing Preserved

- Phase 1: High-visibility acquisition and trust routes.
- Phase 2: Resource and program discovery routes.
- Phase 3: Utility, policy, and system routes.

## Gate Model Preserved

- Inventory regeneration after each phase.
- Hero congruency checks on changed routes.
- Type/lint verification and desktop/mobile spot checks.
