# HANDBOOK-FORM-07 (FORM CP) — Project Stakeholder Photo Release Form

- **Schema:** `form-handbook-cp-client-photo-release-form.json`
- **TOC Code:** FORM CP
- **Chapter:** 7 — Technology & Data Use
- **Owner:** Human Resources
- **Purpose:** Captures a Project Stakeholder's authorization for project photo and media use.

## Fields

| Section                      | Type                | Fields                                                                       |
| ---------------------------- | ------------------- | ---------------------------------------------------------------------------- |
| Client & project information | fieldGrid           | Client / Organization, Project Name, Project Address, Client Rep.            |
| Authorization                | refNote             | Project media-use authorization statement                                    |
| Release Selection            | checkGrid           | Full Release Granted, Internal + Proposal Use Only, Limited / Restricted Use |
| Restrictions or Notes        | narrative           | Free-text restrictions                                                       |
| Restricted-site note         | refNote             | Project-specific media protocol reminder                                     |
| Sign-Off                     | signatures (manual) | Client Representative, MH Construction Representative                        |

## Notes

- For restricted sites, attach the project-specific media protocol and owner approvals.
