# HANDBOOK-FORM-06 (FORM EP) — Employee Photo Release Form

- **Schema:** `form-handbook-ep-employee-photo-release-form.json`
- **TOC Code:** FORM EP
- **Chapter:** 7 — Technology & Data Use
- **Owner:** Human Resources
- **Purpose:** Captures a team member's authorization for company photo and media use.

## Fields

| Section              | Type                | Fields                                                           |
| -------------------- | ------------------- | ---------------------------------------------------------------- |
| Employee information | fieldGrid           | Employee Name, Employee ID, Department / Crew, Supervisor        |
| Authorization        | refNote             | Media-use authorization statement                                |
| Release Selection    | checkGrid           | Full Release Granted, Limited Internal Use Only, Decline Release |
| Revocation           | refNote             | Written-revocation note                                          |
| Sign-Off             | signatures (manual) | Employee, Witness / HR Representative                            |

## Notes

- Revocation requests must be submitted in writing to HR.
