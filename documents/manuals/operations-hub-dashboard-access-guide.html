Route Map & Features

How to Gain Access

Hub Access
Navigate to /hub directly. No credentials required — individual cards may enforce role checks for specific actions.
Dashboard Access
Use the Admin Sign-In modal. Credentials post to /api/auth/admin-login. On success, session cookie is set and user routes to /dashboard. Unauthenticated sessions redirect to /hub.
